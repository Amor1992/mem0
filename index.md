# 记忆索引

> 最后更新: 2026-03-07

## 目录结构

- `categories/` - 按主题分类的关键记忆
  - `工作.md` - 工作相关记忆
  - `生活.md` - 生活琐事与习惯
  - `学习.md` - 学习进度与笔记
  - `育儿.md` - 育儿经验与记录
  - `旅游.md` - 旅行经历与计划
- `daily/` - 每日记录 (格式: YYYY-MM-DD.md)
- `conversations/` - 重要对话摘要

## 使用规则

1. **新增记忆**: 更新对应的类别文件和索引
2. **每日记录**: 在 `daily/` 中记录当天重要事件
3. **重要对话**: 在 `conversations/` 中摘要关键信息
4. **定期整理**: 每周整理一次，将重要内容从daily移到categories

## 快速查找

按时间:
- 最新记录: 查看 `daily/` 最新文件
- 历史记录: 查看 `index.md` 底部时间线

按类别:
- 工作 → `categories/工作.md`
- 生活 → `categories/生活.md`
- 学习 → `categories/学习.md`
- 育儿 → `categories/育儿.md`
- 旅游 → `categories/旅游.md`
